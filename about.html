About MOAP

You've reached the end of computer science

What does it do?

The Mother Of All Programs is a program which automatically designs computer algorithms.

Algorithms are considered by many to be the most crucial part of computer programming. They represent the underlying ideas and theories that describe how a program is going to work. Algorithm design is widely considered to be the most difficult, most time-consuming part of high-level programming.

MOAP automates this process. By taking a formalized statement of an algorithm problem and applying MOAP's problem-solving engine, the program can deliver optimal solutions to a wide variety of algorithm problems in seconds.

How do I use it?

MOAP takes algorithm problems in a special language. Visit our Getting Started page to learn how to use it.
